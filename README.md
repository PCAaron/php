## 什么是集成环境包
> 学习php需安装网页服务器、数据库服务器和php语言核心的解释器。而将这些合在一起的即集成环境包。
常用的集成环境包有AppServ，PHPStudy，APMserv，XAMPP，WAMPServer等（代表的不同集成环境包的名字）。
对入门开发来说，选择集成环境包的原则：1.更新快，版本新；2.操作简单易上手；3.选择项不多
